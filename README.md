# 중고레이더 Vercel MVP

GitHub에 업로드 후 Vercel에서 배포할 수 있는 중고레이더 MVP 프로젝트입니다.

## 배포 순서

1. GitHub에서 새 Repository 생성
2. 이 폴더 안의 파일을 전부 업로드
3. Vercel에서 Add New Project 클릭
4. GitHub Repository 선택
5. Deploy 클릭

## 현재 기능

- 중고레이더 로고
- 통합 검색 UI
- 플랫폼 필터 UI
- 샘플 데이터 기반 검색 결과
- 상세 패널
- 찜 기능
- 저장 검색어
- API 검색 테스트 구조 `/api/search`

## 주의

실제 플랫폼 데이터 연결은 각 사이트 구조/차단 여부에 따라 달라질 수 있습니다.
현재 API는 테스트용 구조입니다.
