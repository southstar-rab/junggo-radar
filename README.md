# 중고레이더 MVP v4 API 테스트 버전

## 핵심 변경
- 검색창에서 실제 `/api/search?q=검색어` 호출
- 실제 연결 결과가 있으면 실제 결과 우선 표시
- 실패하면 샘플 데이터로 fallback
- 화면에 API 연결 상태 표시
- 번개장터 검색 연결 테스트 API 포함

## 반영 방법
1. ZIP 압축 해제
2. GitHub 저장소에 기존 파일을 덮어쓰기 업로드
3. Commit changes
4. Vercel 자동 재배포 확인

## 주의
실제 플랫폼은 크롤링 차단/구조 변경이 있을 수 있어 결과가 안 나올 수 있습니다.
그 경우 화면에는 실패 안내와 샘플 데이터가 표시됩니다.
